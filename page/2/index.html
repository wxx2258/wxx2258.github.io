javascript事件机制

事件流

  1. 事件：我们在网页中的某个操作（有的操作对应多个事件）。例如：当我们点击一个按钮就会产生一个事件。是可以被 JavaScript 侦测到的行为。
  2. 事件触发的三个阶段：
    • 事件捕捉
    • 到达事件位置，触发事件（如果这里既有捕捉事件，也有冒泡事件，则按注册的顺序执行）
    • 事件冒泡，并触发。
